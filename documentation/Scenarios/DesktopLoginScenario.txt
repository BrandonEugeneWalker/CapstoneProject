Employee Login:
1.	An employee loads warehouse application.
2.	An employee enters id and password.
3.	An employee clicks login.
4.	An employee logs in to application.
5.	An employee logs out.

Alternate:
4a: Invalid login: An employee enter incorrect login information, either unknown username or incorrect password, or both.
	.3: An employee is informed that information is incorrect.
	.4: An employee re-attempt to login with correct information.
