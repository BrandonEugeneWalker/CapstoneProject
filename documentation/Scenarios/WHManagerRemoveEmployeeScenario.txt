Manager Remove Employee:
1.	A manager loads warehouse page.
2.	A manager logs in.
3.	A manager clicks manage employees page.
4.	A manager clicks delete on employee row.
5.	A manager is rediected to delete confirmation page.
6.	A manager clicks delete on confirmation page.
7.	A manager has removed the employee.
8.	A manager logs out.

Alternate:
2a: Manager skip login) A manager click manage employees page.
	.3: A manager is redirected to login page.
	.4: A manager cannot access manage employee page without logging in.
6a: Manager doesn't confirm) A manager click back to list on confirmation page.
	.3: A manager is redirected to manage employees page.
